# vue ssr 服务器端渲染例子

*最近在学习vue ssr，将学习过程做的一些例子做了一些整理，汇集于此*

```bash
$ git clone ...
$ cd vue-ssr-exapmple
$ npm install
```

## demo1-基本例子

```bash
$ npm run demo1
```
ssr与服务器集成的基本例子，对应[官方文档](https://ssr.vuejs.org/zh/basic.html)

## demo2-组件生命周期钩子函数

```bash
$ npm run demo2
```
vue组件在服务器端渲染，生命周期钩子函数执行情况，对应[官方文档](https://ssr.vuejs.org/zh/universal.html)
